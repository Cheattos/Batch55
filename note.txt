APa itu Sequalize(tools) = ORM yang membantu kita untuk berkomunikasi ke database

https://sequelize.org/docs/v7/cli/ ikutin langkahnya

npx sequelize-cli model:generate --name User --attributes firstName:string,lastName:string,email:string
ini untuk bikin model baru berdasarkan namanya yaitu User
setaip kita bikin model baru akan selalu otomatis dibikinkan file migration

npx sequelize-cli db:migrate
migration akan diproses oleh sequalize mengtraslate kode kita menjadi kode query yang diterima oleh database

secara otomatis akan ada datanya di database
